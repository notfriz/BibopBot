# TransBot

TransBot es un bot de Discord diseñado para grabar audio en canales de voz y transcribirlo.

## Requisitos

- Python 3.8 o superior
- `ffmpeg` instalado en el sistema
- Credenciales de Google Cloud Speech-to-Text (opcional)
